Magnet

Magnet is a Vanilla JavaScript library and a jQuery plugin to
create gapless, filterable and sortable layouts.

Buy ‚ùØ

Layout Modes

Magnet position items depending on a layout mode. A layout mode, follows its own layout logic.

Grid

Magnet lays out items using a bin-packing algorithm, creating gapless layouts. By default, this is the layout mode used.

Horizontal

Magnet lays out items horizontally. Horizontal layout modes need the container to have a height value:

//Both are valid values
.grid { height: 50%; }
.grid { height: 500px; }
