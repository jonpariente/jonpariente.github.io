About

Magnet is a Vanilla JavaScript library and a jQuery plugin to create gapless, filterable and sortable layouts. Built in the powerful Vanilla JavaScript framework, doesn't require extra libraries. Uses a bin-packing algorithm to create gapless layouts. Features options, methods, events and more to fit any project. Responsive, adapts to any device and screen size. Completed with an extensive documentation and examples.
